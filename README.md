Previsão de Score de Crédito com Machine Learning

Projeto desenvolvido em Python para prever o score de crédito de clientes utilizando Machine Learning.

Tecnologias utilizadas

- Python
- Pandas
- Scikit-Learn
- Random Forest

Descrição

O projeto utiliza dados históricos de clientes para treinar um modelo de Machine Learning capaz de prever o score de crédito de novos clientes.

O modelo identifica padrões entre características dos clientes (idade, salário, profissão etc.) e o score de crédito.

Etapas do projeto

1. Importação dos dados
2. Limpeza e tratamento de dados
3. Conversão de dados categóricos para numéricos
4. Separação entre variáveis de entrada e variável alvo
5. Divisão da base em treino e teste
6. Treinamento do modelo Random Forest
7. Avaliação da acurácia do modelo
